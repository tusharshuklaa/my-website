[VS Code](https://code.visualstudio.com/) is a code editor developed by Microsoft.

### Why VS Code?

VS Code is my go-to tool for coding and development. Its rich ecosystem of extensions and built-in features make it a powerful editor for various programming languages and frameworks.

Free tier covers most needs; paid plans add collaboration features. VS Code is available on multiple platforms, including Windows, macOS, iOS, and Android.
